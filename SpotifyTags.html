Spotify Tags

CSE 115 Software Engineering Web Development Project

Using Agile software development methodology, our team of 5 developed a web application for the Intro to Software Engineering class project. Our product, SpotifyTags, is a tag-based music streaming service. This means that instead of the user filling playlists with songs, the user fills songs with tags. A boolean expression using these tags then filters which songs should be played. For example, selecting "rock AND instrumental" or "sad AND slow BUT NOT classical" only plays the songs that you have tagged that fit the criteria.

My Role

I was a front-end developer for this project, as well as a Scrum Master for a sprint. Going into this project was daunting as I had no prior experience with front-end or the technologies required, like React.js. The Scrum framework allowed us to help each other for our team to be as efficient as possible. It allowed those with more experience with the technologies used to easily guide others when needed. My tasks resolved heavily around building the expression, from writing the logic to sort songs to creating the pop-up to add tags to the expression.
